SECTION 1 — CANONICAL RANK CURVE
=====================================

Metric:
- XP (promotion-only, no decay, no demotion)

Tiers:
- Bronze: 0 – 2,999 XP
- Silver: 3,000 – 7,999 XP
- Gold: 8,000 – 17,999 XP
- Platinum: 18,000 – 34,999 XP
- Diamond: 35,000 – 59,999 XP
- Master: 60,000+ XP

Rules:
- XP never resets
- No seasons yet
- No rank decay
- Rank changes only on XP threshold crossing
